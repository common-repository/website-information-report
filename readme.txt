== Description ==

This plugin displays information such as WordPress version, PHP version, installed plugins (active and inactive), comments, and posts..

1. Version in PHP , WordPress Version 
2. Total number of published posts
3. Total number of comments
   3.1. Total Approved Comments 
   3.2. Total number of pending comments 
4. Total Number of Users
5. Live user status 
 5.1 Away to online(Yellow)
 5.2 Online(Green)
 5.3 Offline(Red)
6. Total Number of Installed Plugins
7. Plugins Installed List
8. Active Plugins List
9. Inactive Plugins List

Kindly let us know your feedback or comments to add more features in this plugin.

== Configuration ==

1. Navigate to Plugins -> Add New in your WordPress admin panel.
2. In the search box, type **Website Information Report** and press the search button.


OR

1. Download and save the **Website Information Report** plugin to your computer.
2. Log in to WordPress and navigate to the Add Plugins page.
3. To upload the zip, click the Upload Plugin button.
4. To install and activate the plugin, click Install Now.
